fun StripeConnectivityRepository.getOnlineStatusFlowable(): Flowable<Boolean>
Content copied to clipboard

Creates an Rx Flowable that emits connectivity status changes from the StripeConnectivityRepository as Boolean flags, where true indicates device is online and false indicates device is offline. Also cleans up afterwards by removing the OfflineListener from the repository when the caller is no longer interested in the stream.
