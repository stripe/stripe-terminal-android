The TerminalApplicationDelegate makes our SDK lifecycle-aware, so that it can respond and react to the various stages of an Android App. It's publicly visible by nature, because users that integrate with our SDK need to call TerminalApplicationDelegate.onCreate and TerminalApplicationDelegate.onTrimMemory from their Application.
